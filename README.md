# Password Generator

Free online secure password generator. Generate random, strong passwords with customizable options. 100% client-side - no data sent to any server.

## Features

- **Customizable Length** - Generate passwords from 4 to 64 characters
- **Character Options** - Toggle uppercase, lowercase, numbers, and symbols
- **Strength Indicator** - Real-time password strength meter
- **One-Click Copy** - Copy generated passwords instantly
- **100% Private** - All generation happens in your browser

## Usage

Visit: https://rayyeung1989.github.io/password-generator/

1. Adjust the password length slider
2. Select character types to include
3. Click "Generate Password"
4. Click "Copy" to copy to clipboard
